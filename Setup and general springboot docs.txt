setup maven project: 
VSCODE:
1. Install extension java extension pack and Spring boot extension pack. 
2. do cntrl+shift+p and type "Spring Initializr: Generate a Maven Project" : This will create a Maven project. 
     (always have the package name starting from com. or org.
3.  the pom.xml file will have all the dependencies for the project. 
      eg: for web project you have to add the following in the <dependencies></dependencies> : This installs all web related thing. 
      <dependency>
    	<groupId>org.springframework.boot</groupId>
    	<artifactId>spring-boot-starter-web</artifactId>
       </dependency>
4. Also make sure you have the following which specifies a springboot as a parent dependency. 
    <parent>
          <groupId>org.springframework.boot</groupId>
           <artifactId>spring-boot-starter-parent</artifactId>
           <version>3.4.0</version>
           <relativePath/> <!-- lookup parent from repository -->
     </parent>

5. For running the application: Install Apache Maven and extract it and then choose a path till bin and add it to the env. variable then inside
    your project where pom.xml file is there go to cmd and type "mvn clean install" and then run the application "mvn spring-boot:run"
